**生成日期**：2026-03-13
**資料來源**：Gemini CLI v0.32.1 × 6 場景並行調查 + Claude Sonnet 4.6 整合分析

## 執行摘要

本報告針對 AMR/AGV 六大應用場景，系統性盤點各場景對電池規格的差異化需求。
核心發現如下：

| 重點發現 | 說明 |
|---|---|
| **場景分歧最大的規格** | 系統電壓（24V vs 800V）、充電倍率（0.5C vs 10C）、認證要求（無 vs ATEX+FDA） |
| **最通用化學體系** | LFP（LiFePO4）覆蓋 S1/S2/S3/S6；NMC 適用 S5；GMP 場景需特規封裝 |
| **業務切入最低門檻** | S1（倉儲）與 S2（重載叉車）：市場成熟、需求規格明確、現有電池廠商可對應 |
| **毛利最高場景** | S4（醫療/GMP）：認證壁壘高、客製化程度高，ASP 為工業標準的 3-5 倍 |
| **IPT 轉型最直接路徑** | S2/S6：大容量深循環電池，與 IPT（感應充電）基礎設施可共存或替代 |

## 二、場景別深度分析

### S1 — 倉儲揀貨 / G2P / P2G / Sortation AMR

**核心定義規格：機會充電 2C–3C**

現代高速倉庫中，AMR 的稼動率是唯一關鍵 KPI。機會充電（Opportunity Charging）使 AMR 在自然閒置時間（訂單排隊、換班 10 分鐘）即可回充 25–30% SOC，徹底消除換電池的人力需求。

**選型決策因子：**
1. **充電倍率 > 能量密度**：G2P 的搬貨距離短，每次移動耗電少，問題在維持 24/7 不停機
2. **LFP 優先於 NMC**：熱穩定性更好（熱失控 270°C vs 210°C），高密度倉庫的安全容忍度要求
3. **SOC 精度 ±2–5%**：影響車隊排程（Fleet Management），防止機器人在路上電力耗盡

**Top 3 供應商**：Inventus Power（PROTRX 系列）、Green Cubes Technology、Flash Battery（Kaitek）

---

### S2 — 中重載搬運 / 自主叉車 / Pallet AMR

**核心定義規格：深循環 + 配重整合**

鉛酸 → 鋰電遷移是本場景最大商機，但電池不只是能源，同時是叉車的**配重（Counterweight）**。換鋰電後往往需要補充鋼板配重。

**Lead-Acid vs LFP 對比：**

| 維度 | 鉛酸（傳統）| LFP（鋰電）|
|---|---|---|
| 能量密度 | 30–40 Wh/kg | 90–120 Wh/kg |
| 充電時間 | 8–12 小時 | 1–2 小時 |
| 維護需求 | 高（補水、均衡充電）| 零（BMS 自動管理）|
| TCO | 低 CAPEX / 高 OPEX | 高 CAPEX / 低 OPEX（3x ROI）|
| 配重功能 | 天然重（鉛密度高）| 需補充鋼板配重 |

**選型決策因子：**
1. **電壓架構**：1.5T 以下 48V；2T–3T 以上 80V（降低 I²R 損耗）
2. **快換 vs 機會充電**：快換（IEC 62840）適合 24/7 多班制；機會充電適合 AMR 路線可預測場景
3. **BMS 高電流**：>200A 持續 / 500A 峰值，需針對感性負載（液壓泵）調整 OCP 邏輯

**Top 3 供應商**：EnerSys（NexSys iON）、Flash Battery（Kaitek）、Flux Power（LiFT Pack）

---

### S3 — 冷鏈 / 戶外跨場域 AMR

**核心定義規格：寬溫域（-30°C~+55°C）+ 智能預熱 BMS**

冷鏈場景最危險的失效模式是**低溫充電鋰鍍（Lithium Plating）**：在 <0°C 時充電會形成金屬鋰枝晶，刺穿隔膜導致內短路，在下次「回溫後充電」時引發熱失控。

**化學體系溫度性能對比：**

| 低溫性能 | LFP | NMC | LTO（鈦酸鋰）|
|---|---|---|---|
| -10°C 容量損失 | 20–25% | 10–15% | <5% |
| -20°C 容量損失 | 40–50% | 25–30% | 10–12% |
| 低溫充電能力 | 需預熱至 +5°C | 需預熱至 0°C | **可在 -30°C 直接充電** |
| 循環壽命 | 3,000–5,000 | 1,000–2,000 | **20,000+** |
| 成本 | 中 | 中高 | **2–3× LFP** |

**LTO 何時值得投資**：24/7 高通量冷凍倉（如生鮮電商前置倉），若等待 45 分鐘預熱需增購 30% 以上機隊數量，LTO 的 TCO 反而更低。

**Top 3 供應商**：Flash Battery（歐洲冷鏈 AMR 部署）、Microvast（LTO 領導廠商）、Green Cube Technology

---

### S4 — 醫療 / 製藥 / GMP 潔淨室 AMR

**核心定義規格：零粉塵封裝 + FDA 21 CFR Part 11 追溯 BMS**

標準工業 LFP 電池**無法直接使用**於 GMP 環境，三大原因：
1. **粉塵生成**：工業等級外殼常有主動風冷或排氣孔，潔淨室禁止顆粒物釋放
2. **數據完整性不符規**：標準 BMS 無加密簽名記錄，不符 FDA 21 CFR Part 11 電子批次記錄要求
3. **外殼材料揮發**：普通 ABS 塑料在 IPA（70% 異丙醇）消毒環境下 30 天後應力開裂，形成細菌陷阱

**GMP 合規的特殊要求：**
- 外殼：316L 不鏽鋼或醫療級 PBT/PC（IPA + VHP 雙重相容）
- 洩漏電流：<100 µA（IEC 60601-1 病患附近要求）
- EMI：Class B 發射限值 + 80MHz–2.7GHz 10V/m 抗擾
- 批次追溯：每顆電芯獨立序號 + 維護記錄 → GMP IQ/OQ/PQ 驗證

**Top 3 供應商**：Inventus Power（U1LiFe 系列）、Accutronics（Ultralife 旗下）、Cleantron（歐洲醫療 AMR 專家）

---

### S5 — 最後一哩 / 戶外配送 / 巡邏 AMR

**核心定義規格：重量能量密度（Wh/kg）極值 + IP67**

NMC 是本場景首選，即使相比 LFP 安全性略低。**重量決定一切**：

```
計算範例（60kg 配送機器人，40km 路程）：
能耗估算：~50 Wh/km（含感測器與計算平台功耗）
所需電量：40km × 50 Wh/km = 2,000 Wh (2 kWh)

NMC（160 Wh/kg 系統級）→ 電池重量：12.5 kg
LFP（100 Wh/kg 系統級）→ 電池重量：20.0 kg

差距 7.5 kg = 越車道、上人行道時需更大馬達與懸掛
```

**NMC 優先於 LFP 的三大理由**：
1. 爬路緣能力：每公斤需要更大馬達 → 能量密度差距在「城市地形敏捷性」上是決定性優勢
2. 低溫性能：NMC 在 -20°C 的容量保留率優於 LFP
3. 體積：LFP 體積幾乎是 NMC 2 倍，壓縮配送空間

**Top 3 供應商**：Inventus Power（U1 NMC 系列）、Green Cubes Technology（Guardian 平台）、Simplo Technology（高量產 21700 NMC 客製）

---

### S6 — 港口 / 重工 / 礦業 超重載 AGV/AMR

**核心定義規格：高壓架構（350–800V）+ 防爆認證**

標準叉車電池（48V/80V）不適用的根本原因是**電流密度**：

```
一台 70 噸港口 AGV 的加速功率需求 ~300 kW

80V 系統：300,000W ÷ 80V = 3,750 A → 電纜粗如手臂，連接器發熱嚴重
700V 系統：300,000W ÷ 700V = ~430 A → 標準汽車級線徑可支撐

I²R 損耗：80V 系統比 700V 系統多耗費 ~47 倍的電纜熱損（同電阻條件）
```

**模組化設計的必要性**：
- 200–600 kWh 電池組無法整體吊裝維修
- 每個模組 <25 kg 以允許人工抽換（維修不需起重機）
- 冗餘架構（Dual-String）：部分電池失效時以 50% 動力「跛行回站」

**Top 3 供應商**：Leclanché（瑞士，港口 AGV 800V G-NMC）、BorgWarner/Akasol（德國，重型商用車液冷系統）、Saft/TotalEnergies（法國，礦業 ATEX 認證 LFP/NMC）

### 3.2 最佳切入策略

**短期（0–12 個月）：S2 叉車鋰電替換**
- 市場驅動力：鉛酸 → 鋰電遷移浪潮，TCO 驅動採購
- 切入方式：與叉車廠（林德、豐田 BT、克朗）合作提供「電池升級方案」
- 現有 IPT 業務協同：叉車已有固定充電站基礎設施，可轉型為「快充站 + 智能 BMS 管理平台」
- 目標客戶：3PL（第三方物流）、電商自動化倉庫、冷鏈物流中心

**中期（12–36 個月）：S1 倉儲 + S6 港口**
- S1：搭上 G2P 系統（Geek+、海康機器人、快倉、極智嘉）的電池模組供應鏈
- S6：進入港口 AGV 充電基礎設施招標（港口往往是 BOT 模式，長期合約）

**長期（36 個月+）：S4 醫療/GMP**
- 進入壁壘最高，但一旦通過認證，替換成本極高（客戶黏性最強）
- 建議先鎖定台灣本地製藥廠（台積電生技園區、晟德、永信）作為驗證場域

### 3.3 現有 IPT 電池業務轉型路徑

```
現有業務          轉型方向                 目標市場
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
IPT 充電基礎設施  →  自動充電站 + BMS 平台  →  S1/S2 倉儲、S6 港口
大容量電池組      →  LFP 叉車電池模組       →  S2 叉車替換市場
電池 BMS          →  醫療合規 BMS           →  S4 GMP 認證電池
無線充電技術      →  S5 配送機器人無線充電  →  S5 最後一哩 AMR
```

**核心建議**：IPT（感應電能傳輸）不是 AMR 電池的對立面，而是**充電基礎設施**的升級方向。
建議定位為「AMR 能量管理系統供應商」，而非單純「電池供應商」：
- 電池模組（BMS）+ 充電站（IPT/有線）+ 車隊充電調度軟體 = 完整解決方案
- 此定位對 S2（叉車）和 S6（港口）客戶的採購說服力最強
